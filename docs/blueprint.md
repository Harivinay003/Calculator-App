# **App Name**: SimpleCalc

## Core Features:

- Input & Operations: Input numbers and select basic arithmetic operations (+, -, *, /).
- Clear Display: Display the current input and the calculation result clearly.
- Calculate Result: Perform arithmetic calculations based on user input.
- Clear: Clear the display to reset the current calculation.

## Style Guidelines:

- Primary color: Soft blue-gray (#A7B2BC) for a calm, user-friendly feel.
- Background color: Light cool gray (#F0F4F7), near-white to maintain simplicity.
- Accent color: Deeper slate-gray (#6B7280) to highlight results, or current operation.
- Font: 'Inter' (sans-serif) for both headings and body text to maintain simplicity and readability.
- Simple, minimalist icons for operation buttons.
- Clean, spacious layout with large, easily tappable buttons.
